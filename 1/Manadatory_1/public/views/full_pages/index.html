Node.js - what is it and what it does

Node.js is a JavaScript runtime powered by the Google V8 Engine. Is it a car then? Not at all! It has similarities to a car, in that it drives the code you write to a compiled state and allows it to achieve its functionality. It actually works more like a road system, having defined rules on how vehicles can get from point A to point B, but not implying that only specific people can drive the cars, or that specific cars can use the roads.

This runtime is used primarily for server programming. Servers are individual computers, or a network of computers that provide access to resources. It can provide access to a website, as it is the case here, or to different other types of data and even files, that other computers can access through a network.

Is Node.js worth it?

First of all, Node.js is a free tool! On top of that, it also has an extensive library of modules that increases the developer's capability to design and write fast and efficient code wich leads to an application with lots of features.
